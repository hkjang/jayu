## 🏗️ 시스템 레이어 아키텍처

시스템은 데이터 수집부터 연산, 의사결정 진화, 외부 연동까지 총 5개의 레이어로 나뉘어 동작하며, 각 레이어는 느슨한 결합(Loose Coupling)을 유지합니다.

```
┌────────────────────────────────────────────────────────────────────────┐
│ 1. 데이터 수집 & 전처리 레이어 (yfinance, pykrx, check_csv, fix_tickers) │
└─────────────────────────────────┬──────────────────────────────────────┘
                                  │ DataFrame (OHLCV)
                                  ▼
┌────────────────────────────────────────────────────────────────────────┐
│ 2. 지표 연산 & 백테스트 레이어 (add_indicators, backtest, calc_metrics) │
└─────────────────────────────────┬──────────────────────────────────────┘
                                  │ Trades Log & Metrics Dictionary
                                  ▼
┌────────────────────────────────────────────────────────────────────────┐
│ 3. 유전 진화 & 최적화 레이어 (run, tournament_select, mutate, meta)    │
└─────────────────────────────────┬──────────────────────────────────────┘
                                  │ best_strategy.json / today_signals.json
                                  ▼
┌────────────────────────────────────────────────────────────────────────┐
│ 4. 외부 인터페이스 & 전송 레이어 (stock_kakao, Kakao API, config.json) │
└─────────────────────────────────┬──────────────────────────────────────┘
                                  │ KakaoTalk Notification / JSON Reports
                                  ▼
┌────────────────────────────────────────────────────────────────────────┐
│ 5. 정적 안정성 & 검증 레이어 (test_simulation.py - 15대 유닛 테스트)   │
└────────────────────────────────────────────────────────────────────────┘
```

## 🐍 모듈별 기능 및 역할 명세 (Role & Responsibility)

### 1. 핵심 유전 진화 및 백테스팅 엔진 (`danta_simulation.py`)
*   **역할**: 시뮬레이션의 중심 브레인. 유전자 알고리즘과 Walk-Forward 백테스트를 결합해 스스로 우수 전략을 진화시킵니다.
*   **상세 기능**:
    - **지표 생성**: pure Pandas 연산으로 고성능 기술 지표(RSI2, ADX, SMA5/200, MACD 등)를 연산.
    - **백테스트 시뮬레이터**: 거래 비용, 동적 슬리피지, 트레일링 스톱, 본전 손절(Break-Even Stop)을 실시간 시뮬레이션.
    - **유전 진화 제어**: tournament 선택 및 변이/교차를 통해 매 회차(500회) 3대 시장 국면별 독립 파라미터를 자율 튜닝.
    - **메타 피드백**: 성공적인 자율 파라미터 셋의 감쇠 가중치 누적 및 샘플링 반영.

### 2. 정적 안정성 검증기 (`test_simulation.py`)
*   **역할**: 코드 수정 시 발생할 수 있는 알고리즘 회귀(Regression)와 런타임 크래시를 방어하는 테스트 프레임워크.
*   **상세 기능**:
    - **지표 검증**: 지표의 NaN 결측치 및 0 나누기(Division by zero) 등의 스트레스 상황 방어 테스트.
    - **전략 흐름 검증**: 래리 코너스 5일선 돌파 청산(`connors_exit`), 본전 스톱 상향 및 청산(`breakeven_exit`) 통합 시나리오 테스트.
    - **수치 안전망 검증**: Sharpe/Sortino/Calmar 비율이 발산(inf)할 때의 클리핑(Clipping) 상한 방어 테스트.
    - **원자적 쓰기 검증**: 저장 중 중단 시 원본 JSON 파일이 손상되지 않는지 검증.

### 3. 실시간 신호 및 알림 허브 (`stock_kakao.py`)
*   **역할**: 매일 지정 시간에 미국/한국 거시 지표 상태와 당일 진입 추천 신호를 사용자 스마트폰으로 카카오톡 전송.
*   **상세 기능**:
    - **시장 모멘텀 진단**: DOW, S&P500, NASDAQ, KOSPI, KOSDAQ 지수의 단기/장기 모멘텀 상태 요약.
    - **시그널 전송 바인딩**: `danta_simulation.py` 완료 시점에 자동 연동 호출되어 `today_signals.json`을 파싱 및 한글화 메시지화하여 전송.
    - **API 방어**: 카카오 토큰 만료 또는 default 한글 토큰 설정 시 인코딩 에러 크래시가 발생하는 현상을 방지하는 예외 방어막 작동.

### 4. 자산 현황 집계기 (`build_portfolio.py`)
*   **역할**: 로컬 CSV 파일에 기재된 244개 종목의 보유량과 현재가를 야후 파이낸스 실시간 시세 및 실시간 원/달러 환율과 연동해 실시간 원화 평가 자산을 실시간 집계.
*   **상세 기능**:
    - **실시간 환율 파싱**: USD/KRW의 실시간 환율을 조회해 원화 자산으로 통합 연산.
    - **포지션 시세 업데이트**: 로컬 CSV 파일 내의 보유 수량 정보와 결합하여 자산 가치 총액 산출.

### 5. 자산 포트폴리오 분석기 (`analyze_portfolio.py`)
*   **역할**: 전체 보유 자산의 섹터별 편중도와 개별 종목 노출 위험도를 분석하는 모듈.
*   **상세 기능**:
    - **섹터 비중도 연산**: YieldMax 배당형 ETF, 레버리지, 반도체 등 각 테마의 비중을 퍼센티지로 요약하여 자산 쏠림 방지 가이드라인 제시.

### 6. 데이터 유틸리티 (`check_csv.py` & `fix_tickers.py`)
*   **역할**: 토스증권 등에서 출력한 CSV 데이터의 구조적 결함을 진단하고, 상장폐지 혹은 티커 변경으로 야후 파이낸스에서 다운로드가 막힌 기형 티커를 자동 보정해주는 도구.

## 🛠️ 유지보수 및 장애 대응 (Troubleshooting)

### 1. 카카오톡 전송 실패 (`Access Token` 만료)
- **증상**: 카카오톡 전송 시 `401 Unauthorized` 에러가 발생하거나 메시지가 오지 않음.
- **조치**: 카카오 개발자 센터에서 액세스 토큰을 재발급받아 `config.json` 내 `"KAKAO_ACCESS_TOKEN"` 값을 최신 토큰으로 교체해 주십시오. 토큰 오류 발생 시 시스템은 자체 Guard Clause에 의해 크래시되지 않고 로그 기재 후 정상 동작을 유지합니다.

### 2. yfinance 데이터 수집 지연 및 실패
- **증상**: 시뮬레이션 기동 시 특정 종목에서 시세 데이터 다운로드 타임아웃 발생.
- **조치**: `danta_simulation.py`에는 **3회 자동 재시도(Retry)** 및 1초 지연 슬립 기능이 내장되어 있습니다. 3회 연속 실패하는 경우 해당 종목은 이번 시뮬레이션 회차에서 무결성 필터에 의해 스킵되며 다음 스케줄 주기에 재시도합니다.
