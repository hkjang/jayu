## 📋 사전 준비 (매일 체크리스트)

### 장 시작 전 (한국시간 오후~저녁)

```
□ 미국 선물 방향 확인 (ES: S&P500 선물, NQ: 나스닥 선물)
□ 필라델피아 반도체 지수(SOX) 전일 종가 방향
□ VIX 수치
   - VIX < 20  → 강세장, 레버리지 단타 유리 ✅
   - VIX 20~30 → 변동성, 포지션 절반 축소 ⚠️
   - VIX > 30  → 단타 중단, 현금 대기 ❌
□ 경제지표/연준 발언 일정 (FOMC, CPI, 고용보고서)
□ 보유 종목 관련 뉴스
□ 시뮬레이션 최적 파라미터 확인 (best_strategy.json)
```

## 🎛️ 종목별 진입 전략

### SOXL — 반도체 3x 레버리지

```
진입 조건:
  ✅ 나스닥 선물 +0.5% 이상
  ✅ SOX 지수 전일 종가 위
  ✅ 장 시작 30분 후 첫 눌림목

최적 파라미터 (시뮬레이션 기준):
  손절: -2%  |  목표: +10%  |  보유: 1일
  RSI: 40~60  |  EMA50  |  거래량 1.2x

타이밍:
  - 갭업 직후 바로 사지 말 것 (30분 대기)
  - 오전 고점 형성 후 눌림목 진입
  - 오후 재차 강세 확인 후 추가 진입 가능
```

### TQQQ — 나스닥 3x 레버리지

```
진입 조건:
  ✅ 나스닥 선물 +0.7% 이상
  ✅ 연준 완화적 발언 또는 경제지표 예상치 상회일

최적 파라미터 (시뮬레이션 기준):
  손절: -5%  |  목표: +12%  |  보유: 1일
  RSI: 55~75  |  EMA10  |  거래량 1.2x

특이사항:
  - 빠르게 치고 빠지는 전략
  - 당일 종가 전 청산 원칙
```

### TSLA — 테슬라 (이벤트 기반)

```
진입 조건:
  어닝 시즌: 발표 5~7거래일 전 매수 (IV 상승 전)
  뉴스 단타: 대형 뉴스 후 첫 눌림목
  일반:     5일 이평선 위 + RSI 55~75

최적 파라미터 (시뮬레이션 기준):
  손절: -3.5%  |  목표: +8%  |  보유: 1일
  RSI: 55~75  |  EMA20  |  거래량 1.5x

주요 이벤트:
  - 어닝 발표 (분기별)
  - FSD 업데이트/마일스톤
  - 납품 수치 발표 (분기별)
  - 일론 머스크 트위터/X 발언
  - 로보택시/Optimus 관련 발표
```

## ⚔️ 손절 원칙 (절대 규칙)

```
규칙 1. 손절선 장 열기 전 설정, 장 중 수정 금지
규칙 2. 손절 도달 즉시 시장가 청산 (지정가 금지)
규칙 3. 하루 최대 손실: 계좌의 -2% (초과 시 당일 거래 중단)
규칙 4. 연속 3번 손절 → 1주일 휴식 (멘탈 리셋)
규칙 5. "본전 오면 팔자" 금지 → 손절선 걸리면 무조건 청산
```

## 💰 포지션 사이징

| 확신도 | 비중 | 조건 |
|--------|------|------|
| 최고 (4가지 모두) | 단타 자금 30% | VIX<20 + 강한 선물 + 뉴스 + 눌림목 |
| 보통 (3가지) | 단타 자금 20% | 일반적 진입 |
| 낮음 (2가지) | 단타 자금 15% | 소극적 진입 |
| 1가지 이하 | **진입 금지** | — |

> **단타 전용 자금 = 전체 포트폴리오의 10~20%**

## 📈 청산 원칙

```
목표 달성 → 즉시 절반 청산, 나머지는 손절선을 본전으로 올림 (트레일링)
보유 3일 경과 → 수익/손실 무관, 재검토 후 청산 검토
거래량 급감 → 모멘텀 소멸, 즉시 청산
뉴스 단타 → 뉴스 소멸과 동시에 청산
```
